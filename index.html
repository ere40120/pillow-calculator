🔥 Firebase 설정
Google 로그인으로 데이터를 여러 기기에 동기화하려면 Firebase 프로젝트가 필요합니다. 아래 단계를 따라 무료로 설정하세요.
1단계 — Firebase 프로젝트 생성
console.firebase.google.com 접속 → 프로젝트 추가 → 이름 입력 (예: pillow-calc) → 계속 → 프로젝트 만들기
2단계 — Google 로그인 사용 설정
프로젝트 콘솔 → Authentication → 시작하기 → Google 선택 → 사용 설정 → 저장
3단계 — Firestore 데이터베이스 생성
콘솔 → Firestore Database → 데이터베이스 만들기 → 테스트 모드 선택 → 위치 선택 → 완료
4단계 — 앱 등록 및 설정 복사
콘솔 홈 → </> 아이콘(웹 앱 추가) → 앱 이름 입력 → 등록 → firebaseConfig 객체 전체 복사
5단계 — 아래에 붙여넣기
const firebaseConfig = { apiKey: "AIza...", authDomain: "your-app.firebaseapp.com", projectId: "your-app", ... };
